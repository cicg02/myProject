프로젝트 소개


이 프로젝트는 건설장비의 고장진단 정보를 3D 모델을 통해서 편리하게 제공할 수 있도록 구성한 App입니다.

현장 작업자의 장비 고장 진단 시간 단축 및 유지보수 효율성 향상되며,

장비 유지보수 담당자, 영업 담당자, 건설 현장 작업자가 이용대상입니다.

주요 타겟 디바이스는 Windows Tablet, Galaxy Tab, iPad입니다.

로그인 화면

이 화면은 로그인 화면입니다.

-로그인(수동) : UnityWebRequest 활용한 HTTP 통신으로 계정 정보를 확인
-로그인(자동) : 서버로 부터 부여받은 Token으로 확인 후, 로그인
-화면 모드 : 창모드, 전체화면 모드
-버전 확인 팝업 : 최신버전이 아닐 경우 앱스토어(현대인프라코어) 연결을 위한 웹사이트 연결
